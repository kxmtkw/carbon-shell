# Keybindings

## System

| Binding | Action |
|---------|--------|
| `Super + F9` | Exit Hyprland |
| `Super + F10` | Power off |
| `Super + F11` | Reboot |
| `Super + F12` | Hibernate |
| `Super + P` | Lock session |
| `Super + X` | Open terminal (Alacritty) |

## Panel

| Binding | Action |
|---------|--------|
| `Super + ↑` | Show panel |
| `Super + ↓` | Hide panel |

## Window and Layout

| Binding | Action |
|---------|--------|
| `Super + V` | Toggle floating |
| `Super + B` | Toggle split |
| `Super + N` | Swap split |
| `Super + M` | Move to root |
| `Super + K` | Kill window |
| `Alt + Tab` | Cycle next window |
| `Alt + Shift + Tab` | Cycle previous window |
| `Super + LMB drag` | Move window |
| `Super + RMB drag` | Resize window |

## Workspaces

| Binding | Action |
|---------|--------|
| `Super + 1` / `Super + ←` | Previous workspace |
| `Super + 2` / `Super + →` | Next workspace |
| `Super + 3` | New workspace (if last is empty) |
| `Super + Shift + 0–9` | Send window to workspace 1–10 |

## Controllers

| Binding | Action |
|---------|--------|
| `Super + Tab` | Window switcher |
| `Super + Q` | App launcher |
| `Super + W` | Network |
| `Super + R` | Runner |
| `Super + T` | Theme |
| `Super + O` | Media player |
| `Super + A` | Clipboard |
| `Super + S` | Screenshot |
| `Super + D` | Power menu |

## Screenshots

| Binding | Action |
|---------|--------|
| `Print` | Active screen |
| `Super + Print` | Active window |
| `Super + Shift + Print` | Region select |

## Media & Brightness

| Binding | Action |
|---------|--------|
| `Vol+` / `Vol-` | Volume up/down (+5/-5) |
| `Mute` | Toggle mute |
| `MicMute` | Toggle mic mute |
| `Bri+` / `Bri-` | Brightness up/down (+10/-10) |
| `Play` / `Pause` | Play/pause |
| `Next` / `Prev` | Next/previous track |

## Gestures (3-finger)

| Gesture | Action |
|---------|--------|
| Swipe left / right | Switch workspace |
| Swipe up / down | Toggle floating window |
| Pinch | Toggle fullscreen |
