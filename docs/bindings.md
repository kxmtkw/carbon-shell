| Keybind | Action |
|---|---|
| **>> General <<** | |
| `Super + F9` | Exit Hyprland |
| `Super + F12` | Reboot |
| `Super + F1` | Lock session |
| `Super + T` | Open terminal |
| **>> Layout <<** | |
| `Super + V` | Toggle floating |
| `Super + B` | Toggle split |
| `Super + N` | Swap split |
| `Super + M` | Move to root |
| `Alt + Tab` | Cycle next window |
| `Alt + Shift + Tab` | Cycle previous window |
| **>> Workspaces <<** | |
| `Super + 1` | Previous workspace |
| `Super + 2` | Next workspace |
| `Super + 3` | Create new workspace |
| `Super + ←/→` | Switch workspace |
| `Super + Shift + 1-0` | Move window to workspace 1–10 |
| **>> Windows <<** | |
| `Super + Mouse1` | Move window |
| `Super + Mouse2` | Resize window |
| `Super + K` | Kill active window |
| **>> Controllers <<** | |
| `Super + Tab` | Windows overview |
| `Super + Q` | App launcher |
| `Super + W` | Wifi menu |
| `Super + R` | Run menu |
| `Super + I` | Theme switcher |
| `Super + O` | Media player |
| `Super + P` | Power menu |
| `Super + A` | Clipboard |
| `Super + S` | Screenshot menu |
| **>> Screenshots <<** | |
| `Print` | Screenshot monitor |
| `Super + Print` | Screenshot window |
| `Super + Shift + Print` | Screenshot region |
| **>> Media <<** | |
| `XF86AudioRaiseVolume` | Volume +5% |
| `XF86AudioLowerVolume` | Volume -5% |
| `XF86AudioMute` | Toggle mute |
| `XF86AudioMicMute` | Toggle mic |
| `XF86MonBrightnessUp` | Brightness +10% |
| `XF86MonBrightnessDown` | Brightness -10% |
