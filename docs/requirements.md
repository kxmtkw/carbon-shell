
# Requirements

These include the dependancies the shell needs to function. If using `Arch Linux`, these packages will be automatically installed by the installation script.

The packages listed are the `Arch Linux` version. If you are using some other distribution, then please translate these packages before installation.

## Core Packages

These listed packages are essential for the shell to work. 

```bash
hyprland \
hypridle \
hyprlock \
hyprpolkitagent \
hyprshot \
hyprsunset \
archlinux-xdg-menu \
xdg-desktop-portal \
brightnessctl \
cliphist \
playerctl \
quickshell \
rofi \
swww \
qt5-imageformats \
qt5-multimedia \
qt5-svg \
```

## App Packages

These packages are for the essential desktop apps. These inlcude the bare minimum packages from KDE for a functioning desktop. 

These apps include:

- `Terminal` alacritty
- `Archive Manager` ark
- `Calendar` calindori
- `File Manager` dolphin
- `Disk Usage Analyzer` filelight
- `Image Viewer` gwenview
- `Media Player` haruna
- `Hyprland Settings` hyprviz
- `ISO Image Writer` isoimagewriter
- `Camera` kamoso
- `Calculator` kcalc
- `Clock` kclock
- `Torrent Client` ktorrent
- `Network Manager GUI` nm-connection-editor
- `GTK Appearance Settings` nwg-look
- `Document Viewer` okular
- `Partition Manager` partitionmanager
- `System Monitor` plasma-systemmonitor


Needed packages: 

```bash
alacritty \
ark \
calindori \
dolphin \
dolphin-plugins \
filelight \
gwenview \
haruna \
hyprviz \
isoimagewriter \
kamoso \
kcalc \
kclock \
ktorrent \
nm-connection-editor \
nwg-look \
okular \
partitionmanager
plasma-systemmonitor
```

```bash
kconfig \
kde-cli-tools \

kcoreaddons \

```
